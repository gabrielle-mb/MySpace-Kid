# MySpace-Kid

Before APIs and integrations, there was MySpace.

This folder is a tribute to the original “self-taught engineers” era:
- embedding music with questionable HTML
- changing profile colors with hex codes we barely understood
- breaking entire layouts and fixing them through trial and error

Honestly… not that different from debugging APIs today.

---

### What's in here

- 🎵 How to embed music on your profile  
- 🎨 Color cheat codes (the OG design system)  
- 🧩 Profile layout hacks  

---

### Why this is here

Because I learned a lot of my problem-solving skills by:
- experimenting
- breaking things
- figuring out how to fix them

Same approach, different tools.
